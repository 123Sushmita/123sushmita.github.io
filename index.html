Credit Card Fraud Detection Project

This project focuses on identifying fraudulent transactions using data analysis techniques. It combines Python, SQL, and Tableau to analyze patterns and build a fraud scoring system.

View Full Project on GitHub

Project Workflow

Step 1: Data Understanding

Analyzed dataset features such as transaction amount, hour, merchant category, device trust score, and fraud label. Observed fraud cases are rare.

Step 2: Python Analysis

Used Pandas for data cleaning and Matplotlib/Seaborn for visualization. Identified patterns such as high transaction amounts and high velocity in fraud cases.

Step 3: SQL Fraud Scoring

Created a fraud scoring system using SQL CASE statements:

CASE WHEN amount > 2000 THEN 2 WHEN foreign_transaction = 1 THEN 2 WHEN location_mismatch = 1 THEN 1 WHEN device_trust_score < 0.3 THEN 2 ELSE 0 END AS fraud_score

This helped rank transactions based on risk level.

Step 4: Tableau Visualization

Built dashboards to visualize fraud patterns across multiple dimensions.

Step 5: Key Insights

  • High-value transactions are more likely fraudulent
  • Foreign transactions increase fraud probability
  • Low device trust score is a strong indicator
  • High transaction velocity signals suspicious behavior
